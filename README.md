# Employee Management System (EMS)

A high-performance, modular Desktop Application built using **JavaFX** and strict **Object-Oriented Programming (OOP)** structural architectures.

## 🚀 Core Features
- **Secure System Login Interface**: Admin access gating panel with error handlers.
- **Dynamic Employee Profiles**: Create, update, and drop active employee files dynamically.
- **Live Input Validation Validation Interceptors**: Real-time regex data format filtering for IDs, salaries, and text inputs.
- **Premium Custom UI & Styling**: Bespoke CSS customization overlays featuring premium dark purple themes and smooth row highlight/selection feedback components.

## 🛠️ OOP Principles Implemented
- **Inheritance & Abstraction**: Utilizes an abstract parent base `Person` class extended by a concrete `Employee` subclass to optimize data models and code reuse.
- **Encapsulation**: Strict enforcement of access security by encapsulating member fields behind private modifiers and safe access mutators (getters/setters).
- **Polymorphic Architecture**: Implements runtime polymorphism via overridden method behaviors (`getRoleDescription()`).
- **Data Persistence**: Utilizes Object State Serialization markers (`Serializable`) to seamlessly stream, capture, and load workspace records locally.
